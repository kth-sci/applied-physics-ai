## What you need to do before May 8

**1. Watch your inbox for a Claude Team invitation.** You will receive a separate email from Anthropic in the next day or two inviting you to join the *KTH Applied Physics* Claude Team. Please accept it before the tutorial.

**2. Install Claude Code.** Once you have accepted the invitation, install Claude Code on your laptop:
- **macOS / Linux:** open Terminal and run
  ```
  curl -fsSL https://claude.ai/install.sh | bash
  ```
- **Windows:** download the [Desktop App](https://claude.ai/api/desktop/win32/x64/exe/latest/redirect) (easiest), or follow the [PowerShell instructions](https://kth-sci.github.io/applied-physics-ai/ai-agents.html).

If you get stuck, do not worry — we will help you get it working at the start of the session.

**3. Make sure you can log in to ChatGPT** with a personal account (or a KTH license if you have one).

**4. Optional warm-up reading** (10–15 minutes each):
