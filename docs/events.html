Coming May 8, 2026

APHYS AI Agent Tutorial

May 8, 2026 13:00 -- 17:00 CET APHYS Department (room TBD) Wei Ouyang & Jonas Sellberg

This is your first step into using AI agents for real work. In four hands-on hours, you will:

First Half: Learn the Tools

  • Set up ChatGPT with your KTH account
  • Master prompt engineering basics (the cheat sheet)
  • Install and configure Claude Code
  • Understand the difference between chat and agents

Second Half: Apply to Your Work

  • Use AI for research tasks: data analysis, code generation
  • Use AI for teaching: create materials, review student work
  • Use AI for productivity: draft text, automate workflows
  • Share results and discuss experiences
Preparation: Bring your laptop. No coding experience required -- the AI does the coding for you. If you want to get a head start, read the ChatGPT Quick-Start Guide and the AI Agents introduction on this site.

The skills you build here will be directly applied at the Hackathon on May 26 and all subsequent events. This is the foundation.
