Before you arrive — setup

Please complete these two steps before Thursday. It only takes 10–15 minutes. If you get stuck, arrive a few minutes early — helpers will be there.

1

Create a ChatGPT account (free)

Go to chat.openai.com and sign up with your email. The free tier is enough for today's exercises.

Already have one? You're set.

2

Install Claude Code

You have been granted a KTH Applied Physics Claude Team seat. Install Claude Code with one command:

# macOS / Linux / WSL
curl -fsSL https://claude.ai/install.sh | bash

# Windows (PowerShell)
irm https://claude.ai/install.ps1 | iex

Then start it and log in with your KTH email (the one you registered with):

claude
# Follow the login prompt — it opens a browser tab
Don't have a Claude seat yet? Request one here — approval usually happens within 24 hours.
No laptop? Borrow one from a colleague, or you can follow along and do all exercises on claude.ai in your browser — no installation needed.
