What Is This Initiative?

A department-wide effort to help Applied Physics faculty adopt AI tools — starting with what works today.

The Department of Applied Physics at KTH is launching a series of hands-on activities — tutorials, hackathons, and workshops — designed to help you integrate AI into your research, teaching, and daily work. This is not a machine learning course. It is a practical guide to using AI agents as tools that amplify your scientific expertise.

Our starting point: Two tools that cover the full spectrum. ChatGPT for quick, conversational AI tasks (KTH has procured licenses). Claude Code for autonomous AI agents that can write code, analyze data, and build tools for you.

What Can AI Do for You?

Real capabilities for researchers, educators, and academic professionals.

Research

Analyze spectroscopy/microscopy data, build ML models, automate pipelines, conduct literature reviews, prototype methods.

Teaching

Create course materials, provide feedback on student work, redesign assignments for the AI era, generate exercises with solutions.

Productivity

Draft grant sections, translate legacy MATLAB/Fortran code, create financial plans, automate workflows, build personal tools.
